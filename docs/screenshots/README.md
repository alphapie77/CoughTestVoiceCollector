# Screenshots Directory

This directory contains screenshots for the README Features Showcase section.

## Required Screenshots

Add the following screenshots to this folder:

1. **home.png** - Landing page showing navigation and main features
2. **recording.png** - Recording interface with audio capture controls
3. **analytics.png** - Analytics dashboard with statistics and charts
4. **recordings.png** - Browse recordings page with data table
5. **admin.png** - Django admin panel interface

## How to Add Screenshots

1. Run your application (`scripts\start_servers.bat`)
2. Navigate to each page in your browser
3. Take screenshots (Windows: Win+Shift+S, Mac: Cmd+Shift+4)
4. Save them in this folder with the exact names listed above
5. Recommended resolution: 1920x1080 or 1280x720
6. Format: PNG (preferred) or JPG

## Tips for Good Screenshots

- Use a clean browser window (no extra tabs/bookmarks visible)
- Show the full interface with sample data
- Ensure good contrast and readability
- Crop to show only the relevant application area
- Consider adding sample recordings/data for better visuals

Once you add the screenshots, they will automatically appear in the README's Features Showcase section.
